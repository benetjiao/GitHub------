## 继续GitHub客户端操作
1. 在GitHub客户端 options...选项中 default shell 中，Custom 选项中选择自己编辑器的绝对位置
2. 在repository 中右键自己的 项目 ，选择Open in Git Shell 就可以使用自己选择的编辑器进行编辑了
